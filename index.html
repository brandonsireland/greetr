Greetr Library

Greetr is a simple Greetings library that supports jQuery. It contains 4 different Greets in 6 languages.

Greetr takes 4 inputs: First name, Last name, language and Message Type.

Using Greetr is as simple as using the G$(firstname, lastname, language).

Try out different greetings!

Methods

Greetr has a few methods to use. fullName(), formalGreeting(), informalGreeting(), foulGreeting(), and normalGreeting()

Furthermore, Greetr has chainable methods such as log(), setLang(), setMsgType(), greet() and HTMLGreet().
